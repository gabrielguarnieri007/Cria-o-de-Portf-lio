# Portfólio pessoal - Gabriel

Projeto de portfólio desenvolvido com HTML, CSS e JavaScript.

## Como abrir no VS Code

1. Extraia o arquivo ZIP.
2. Abra a pasta `portfolio-gabriel` no VS Code.
3. Clique com o botão direito no arquivo `index.html`.
4. Selecione **Open with Live Server**.

## O que o projeto possui

- Página inicial com nome, imagem representativa e apresentação pessoal.
- Seção Sobre com biografia, interesses e objetivos.
- Seção de habilidades com HTML, CSS, JavaScript, APIs, GitHub e Responsividade.
- Seção de projetos com dois projetos e imagens.
- Seção de contato com e-mail, GitHub e LinkedIn.
- Layout organizado, navegação funcional e design responsivo.
